## 1. Getting Started

### Logging In

1. Navigate to your Plannetic portal URL (e.g., `plannetic.com`)
2. Enter your email and password
3. Complete 2FA verification if enabled
4. You'll land on the Dashboard

**Tip:** Bookmark your dashboard URL for quick access.

### Dashboard Overview

The Dashboard provides an at-a-glance view of your practice:

- **Client Statistics** - Total clients, vulnerable clients, pending reviews
- **Assessment Progress** - Completion rates for ATR, CFL, and Suitability
- **Upcoming Reviews** - Clients due for annual review
- **Recent Activity** - Latest actions and updates
- **AI Insights** - Intelligent recommendations and alerts

## 2. Managing Clients

### Adding a New Client

1. Click **"+ New Client"** from the sidebar or use `Ctrl+N`
2. Complete the wizard steps:
   - **Personal Details** - Name, date of birth, NI number
   - **Contact Information** - Email, phone, address
   - **Financial Profile** - Employment, income, assets, liabilities
   - **Review** - Verify all information
   - **Confirm** - Save the client

### Client List

The Clients page shows all your clients with:

- **Statistics Cards** - Total clients, vulnerable clients, pending assessments
- **Search & Filter** - Find clients by name, email, or status
- **Sorting** - Sort by name, date added, or review status
- **Quick Actions** - View, edit, or start assessment

### Client Detail Page

Click on any client to view their full profile with tabbed navigation:

| Tab | Contents |
|-----|----------|
| **Overview** | Personal details, status, vulnerability indicators |
| **Financial** | Income, expenditure, assets, liabilities, net worth |
| **Assessments** | ATR, CFL, Suitability results and history |
| **Documents** | Uploaded and generated documents |
| **Communications** | Interaction history and notes |
| **Reviews** | Review schedule and history |

### Client Financials

The Financial tab provides comprehensive wealth analysis:

- **Wealth Breakdown Chart** - Visual representation of assets
- **Income vs Expenditure** - Monthly cash flow analysis
- **Net Worth Calculation** - Assets minus liabilities
- **Financial Insights** - AI-powered recommendations

### Tracking Vulnerable Clients

Plannetic helps identify and track vulnerable clients:

- **Vulnerability Indicators** - Health, life events, capability, resilience
- **Support Flags** - Mark clients requiring additional support
- **FCA Compliance** - Consumer Duty vulnerability tracking
- **Vulnerability Register** - Centralised tracking in Compliance module

## 3. Conducting Assessments

### Assessment Dashboard

Navigate to **Assessments** to see:

- All client assessments
- Completion status
- Assessment scores
- Historical results

### ATR (Attitude to Risk) Assessment

The ATR questionnaire measures psychological tolerance for investment risk:

1. Select a client
2. Click **"Start ATR Assessment"**
3. Complete the 10-question questionnaire
4. Review the calculated risk score (1-10)
5. Save the assessment

**Risk Categories:**
- 1-3: Cautious
- 4-5: Balanced
- 6-7: Moderately Adventurous
- 8-10: Adventurous

### CFL (Capacity for Loss) Assessment

CFL measures financial ability to absorb losses:

1. Select a client
2. Click **"Start CFL Assessment"**
3. Enter financial data (income, expenditure, assets, liabilities)
4. System calculates capacity score
5. Review and save

**Important:** The lower of ATR and CFL determines the recommended investment approach.

### Suitability Assessment

The comprehensive suitability assessment for FCA COBS 9.2 compliance:

1. **Personal & Financial Situation** - Auto-populated from client profile
2. **Investment Objectives** - Goals, time horizons, priorities
3. **Risk Profile Summary** - ATR/CFL scores with interpretation
4. **Recommendation & Rationale** - Proposed solution with justification
5. **Alternatives Considered** - Why other options were not recommended

**AI Assist:** Use the AI auto-complete feature to draft recommendation rationales.

### Investor Persona Assessment

Determine the client's investor personality type:

1. Select a client
2. Navigate to **"Persona Assessment"**
3. Complete the behavioural questionnaire
4. Review the persona result:
   - **Guardian** - Security-focused, prefers stability
   - **Achiever** - Goal-oriented, balanced approach
   - **Adventurer** - Growth-focused, comfortable with volatility
   - **Innovator** - High conviction, concentrated positions

## 4. Financial Analysis Tools

### Cash Flow Modelling

Create detailed cash flow projections:

1. Navigate to **Cash Flow**
2. Select a client
3. Enter parameters:
   - Current portfolio value
   - Regular contributions/withdrawals
   - Expected growth rate
   - Inflation assumptions
   - Time horizon
4. Generate projection chart
5. Adjust scenarios as needed

**Features:**
- Multi-scenario comparison
- Inflation adjustment
- What-if analysis
- Sustainability analysis (how long funds will last)

### Monte Carlo Simulation

Run probabilistic retirement projections:

1. Navigate to **Monte Carlo**
2. Enter retirement parameters:
   - Current age and retirement age
   - Target retirement income
   - Existing pension/investment funds
   - Regular contributions
3. Configure asset allocation (auto-suggested based on risk profile)
4. Run simulation (1,000 scenarios)
5. Review results:
   - **Success Probability** - Target >85% for confidence
   - **Median Outcome** - 50th percentile projection
   - **Confidence Bands** - 10th to 90th percentile range

### Stress Testing

Analyse portfolio resilience under adverse market conditions:

1. Navigate to **Stress Testing**
2. Select a client portfolio
3. Choose stress scenarios:
   - **2008 Financial Crisis** - Severe market downturn
   - **2020 COVID Crash** - Rapid decline and recovery
   - **Rising Interest Rates** - Bond market stress
   - **Stagflation** - High inflation, low growth
   - **Custom Scenario** - Define your own parameters
4. Run analysis
5. Review results:
   - **Portfolio Impact** - Projected loss under each scenario
   - **Resilience Score** - Overall portfolio robustness
   - **Recovery Time** - Estimated time to recover
   - **Recommendations** - Suggested portfolio adjustments

## 5. Reports & Documents

### Generating Reports

1. Navigate to **Reports**
2. Select report type:
   - Suitability Report
   - ATR Summary
   - Cash Flow Projection
   - Monte Carlo Analysis
   - Client Dossier (comprehensive)
3. Select client
4. Click **"Generate Report"**
5. Review and download PDF

### Document Management

The **Documents** module provides:

- **Document Vault** - Secure storage for all client documents
- **Upload** - Drag-and-drop document upload
- **Categories** - Organise by type (ID, contracts, reports)
- **Version History** - Track document versions
- **Search** - Find documents across all clients

### Digital Signatures

Send documents for electronic signature:

1. Generate or upload a document
2. Click **"Send for Signature"**
3. Add signature fields
4. Send to client email
5. Track status: Sent > Viewed > Signed
6. Download signed document

**Integration:** OpenSign provides secure, legally-binding signatures.

## 6. Compliance

### Compliance Dashboard

The Compliance module provides regulatory oversight:

- **Overall Compliance Score** - Health of your compliance posture
- **Outstanding Actions** - Items requiring attention
- **Recent Activity** - Compliance-related actions
- **Regulatory Alerts** - FCA updates and deadlines

### QA & File Reviews

Conduct quality assurance reviews:

1. Navigate to **Compliance > QA Reviews**
2. Select files for review
3. Complete review checklist
4. Record findings
5. Assign actions if needed
6. Mark as complete

### Compliance Registers

Maintain required regulatory registers:

| Register | Purpose |
|----------|---------|
| **Complaints Register** | Log and track client complaints |
| **Breaches Register** | Record compliance breaches |
| **Vulnerability Register** | Track vulnerable client support |
| **Gifts & Hospitality** | Record gifts received/given |

### Consumer Duty Tracking

Monitor Consumer Duty obligations:

- **Products & Services** - Assess client outcomes
- **Price & Value** - Fair value assessments
- **Consumer Understanding** - Communication effectiveness
- **Consumer Support** - Support quality metrics

### AML/CTF Compliance

Anti-Money Laundering controls:

- **Client Verification** - KYC documentation
- **Risk Assessment** - AML risk scoring
- **Suspicious Activity** - SAR recording
- **Training Records** - Staff training compliance

## 7. Client Reviews

### Scheduling Reviews

Set up annual review schedules:

1. Navigate to **Reviews**
2. View review calendar
3. Click **"Schedule Review"**
4. Select client
5. Set review date
6. Add notes/agenda items
7. Save

### Review Dashboard

The Reviews page shows:

- **Upcoming Reviews** - Next 30 days
- **Overdue Reviews** - Past due date
- **Completed Reviews** - This year's completed reviews
- **Review Calendar** - Monthly view

### Conducting Reviews

1. Select a scheduled review
2. Review client's:
   - Current circumstances
   - Portfolio performance
   - Risk profile changes
   - Objectives progress
3. Update assessments if needed
4. Record review notes
5. Schedule next review
6. Mark as complete

### Review Reminders

Automatic reminders:

- 30 days before review due
- 7 days before review due
- On review due date
- Overdue alerts

## 8. Communication

### Logging Communications

Record all client interactions:

1. Navigate to client profile > **Communications** tab
2. Click **"Log Communication"**
3. Select type: Call, Email, Meeting, Letter
4. Enter details:
   - Date and time
   - Duration (for calls/meetings)
   - Summary of discussion
   - Action items
5. Save

### Notification Centre

Access the notification bell in the header:

- **New Messages** - Client communications
- **Task Reminders** - Upcoming deadlines
- **Compliance Alerts** - Regulatory notifications
- **System Updates** - Platform announcements

### Messages

The **Messages** module provides:

- **Inbox** - Received messages
- **Sent** - Sent communications
- **Templates** - Reusable message templates
- **Bulk Messaging** - Send to multiple clients
