# Plannetic User Guide - Design Brief

**For AI Designer / Graphic Designer**

---

## Project Overview

Create a visually stunning, concise user guide for **Plannetic** - a financial advisory platform for Independent Financial Advisors (IFAs) in the UK.

**Target Audience:** Financial advisors who need a quick visual reference
**Tone:** Professional, modern, trustworthy
**Length:** 8-12 pages (one page per section)
**Format:** PDF, printable A4 or digital

---

## Icon Style Guide

**Recommended Icon Style:**
- Line icons (not filled)
- Rounded corners
- 2px stroke weight
- Consistent sizing (24x24 or 32x32)
- Teal colour or dark navy

**Icon Libraries to Consider:**
- Lucide Icons (open source)
- Heroicons (Tailwind)
- Phosphor Icons
- Feather Icons

**Custom Icons Needed:**
- Risk scale (1-10 gradient)
- Monte Carlo fan chart
- Compliance gauge
- Consumer Duty pillars
